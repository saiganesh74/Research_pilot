## Setup & Installation  
1. Clone the repo 
```bash
git clone https://github.com/saiganesh74/Research_pilot.git
cd Research_pilot
```
2. Install Deps
  ```bash
  npm install
  ```
3. Run Locally
  ```bash
  npm run dev
  ```
4. Setup ENV folder
  ```env
  touch .env
  GEMINI_API_KEY= YOUR_API_KEY
  SERPAPI_KEY= YOUR_SERP_API_KEY

  ```
